# Wasp-Atlas-Explorer

`Wasp-Atlas-Explorer` is a React application for browsing and interacting with modular building systems powered by `webwaspjs`.

## Install

```bash
npm install
```

## Quick start

```bash
npm run dev
```

The app reads dataset catalog data from the public Wasp Atlas repository and renders interactive assemblies in the browser.

## Development

Requirements:

- Node.js 18+
- npm 9+

Commands:

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run typecheck`
- `npm test`
